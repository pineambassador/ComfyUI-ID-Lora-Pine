Release Notes: ID-LoRA Video v0.3a (Temporal Identity Update)



V0.3a

Major bug fix impacting video and portrait preservation in ‘refiner’ mode.

Workflow updates

  1. New 3-stage workflow Composition-Reinforce-Refine

  2. Introduce workflow and utilities for new (alternate) portrait injection methodology (hard modified frame injection) . Portrait injection is normally implemented as a reference vs a residual frame in the outut



V0.3 Executive Summary

This update introduces the "Identity-First" workflow, focusing on high-fidelity likeness retention through long-form video generations. By moving away from static frame-seeding and toward a Latent Vacuum and Temporal Fading architecture, this version eliminates first-frame freezing and reduces identity drift during complex motion (e.g., profile turns).

New Core Features

1. The "Soft Anchor" Seeding System

  • Variable Noise Masking: The strength slider now directly controls the Noise Mask for Frame 0. Setting strength to 0.90 allows 10% noise permission, enabling the motion module to begin animating on the very first frame while maintaining 90% structural adherence.

  • VAE Scaling Handshake: Automatic detection and correction of LTX2.3 latent distributions (using the 0.18215 scaling factor) to prevent the sampler from "clobbering" injected frames.

2. Multi-Stage Identity Interleaving

  • Interleaved Scene Anchors: Support for reinforcing the environment/scene at specific intervals (e.g., frames 12, 48, 96) without hard-locking the motion.

  • Portrait Handover Logic: New support for multiple identity anchors (e.g., Frontal @ Frame 24 and Profile @ Frame 60) with a customizable Linear Fade. This creates a "Latent Gradient" that guides the model through head turns without "identity popping."

3. Refiner Stage Optimization (The "Correction" Pass)

  • Likeness-Priority Routing: Added a Refiner Mode that prioritizes Portrait Injection over Base-pass artifacts.

  • Optimized Sampler Stack: Recommendation of DPMPP_2M with sgm_uniform for the Refiner pass, providing high-precision sharpening and stable likeness at low denoise (0.20).

  • Granular CFG Balancing: Decoupled Global, Portrait, and Audio CFG to allow "Quiet" global guidance (1.8-2.0) while maintaining "Loud" Identity guidance (4.0+) for facial correction.

Technical Breakdown: IDLoRAPrepareVideo Node

  • Input strength: Links directly to noise_mask ($1 - strength$) for Frame 0.

  • Input fade_frames: Controls the temporal blend length of portrait injections.

  • Output: Returns a synchronized LATENT and NOISE_MASK pair compatible with advanced LTX schedulers.
